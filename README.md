# Casamento — Mari & Davi

Site estático do convite de casamento.

## Publicar no GitHub Pages

1. Faça commit e push dos arquivos da raiz (`index.html`, `styles.css`, `assets/`, `.nojekyll`).
2. No repositório no GitHub: **Settings** → **Pages**.
3. Em **Source** escolha **Deploy from a branch**.
4. **Branch**: `master` (ou `main`, conforme o seu repo). **Folder**: `/ (root)`.
5. Salve. Em alguns minutos o site ficará em:
   - `https://davibatistaoliv.github.io/casamento/`

## Desenvolvimento

O código-fonte fica em `src/`. Para publicar, copie o conteúdo de `src/` para a raiz (ou use apenas os arquivos da raiz após a primeira cópia).
